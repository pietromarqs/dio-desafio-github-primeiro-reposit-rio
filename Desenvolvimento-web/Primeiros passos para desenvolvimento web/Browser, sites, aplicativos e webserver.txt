Browser (navegador): Interprete de linguagens de programação (LP) (utiliza Cache e cookies)

Site:	Página da internet 
		D/XHTML caindo em desuso

Aplicativo: Um software que é executado no navegador
		Hoje em dia, qase não existe diferença entre site e aplicativo

E-commerce: Eletronic Commerce (comércio eletrônico)
		O sistema de pagamento pode ser exterior ao site
			Ex.: PicPay, Pagseguro, Paypal, Mercado pago, etc

Web-server:
	Estático: Servidor físico  
	Dinâmico: Softwares que estão presentes no servidor físico
		Arquivos	(file server)
		Aplicações	(application server)
		Banco de dados	(database)
		Tudo junto e misturado


Web-service: Interface disponível para fazer requisições e consultas em banco de dados inacessíveis
		Ex.: Correios, Gov
